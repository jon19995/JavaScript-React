Все команды для GIT начинаются с git


1. git init  - Указываем гиту, за какой папкой необходимо следить 

2 - создаем конфиг, указываем под каким пользователем мы работаем:

git config --global user.name "Evgenii"
git config --global user.email jon-95@list.ru

3. добавляем все файлы в гит  -  git add -A

4. Создаем коммит (точку сохранения)  -  git commit -a -m"first commit"  В "указываем описаниесохранения, чтоб понять, что изменилось"

5.Чтоб посмотреть все сохраненные точки (коммиты) -  git log

6. Чтоб посмотреть статус git, есть ли новые/измененные файлы и т.д.  -  git status
