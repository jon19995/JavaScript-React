Все команды для GIT начинаются с git


1. git init  - Указываем гиту, за какой папкой необходимо следить 

2 - создаем конфиг, указываем под каким пользователем мы работаем:

git config --global user.name "Evgenii"
git config --global user.email jon-95@list.ru

3. добавляем все файлы в гит  -  git add -A

4. Создаем коммит (точку сохранения)  -  git commit -a -m"first commit"  В "указываем описаниесохранения, чтоб понять, что изменилось"

5.Чтоб посмотреть все сохраненные точки (коммиты) -  git log

6. Чтоб посмотреть статус git, есть ли новые/измененные файлы и т.д.  -  git status

Подключение к гитхабу:

1. git remote add origin https://github.com/jon19995/JavaScript-React.git  - remote add(удаленный репозиторий), origin - название, можно выбрать любую и ссылка на репозиторий

Теперь локальный репозиторий на ПК связан с репозиторием на гитхабе.

2. чтоб отправить репозиторий с ПК на гитхаб: git push -u origin master

Чтобы скачать репозиторий с гитхаба себе на ПК:

1.сначала выходим из нашей папки  -  cd ..

2. Переходим в нужную папку - cd work

3. Клонируем репозиторий: git clone https://github.com/jon19995/JavaScript-React.git project

ссылка с гитхаба, project - название папки в которую будет скачан проект
